== Overview

This handbook is generated from verified MockMvc documentation snippets and the live OpenAPI contract exposed by the application.

Base URL:
`http://localhost:8080`

Response envelope:
JSON endpoints return `success`, `message`, `data`, and `timestamp`.

Timestamp format:
ISO-8601 local date-time, for example `2026-04-14T21:30:00`.

Content types:
Most routes use `application/json`. `POST /people/{id}/cv` consumes `multipart/form-data`. `GET /people/{id}/cv` returns a binary download with `application/octet-stream`.

Error model:
Request body validation, method-parameter validation, missing parameters, multipart part failures, malformed JSON, resource-not-found cases, and conversion failures all return the standard JSON envelope.
